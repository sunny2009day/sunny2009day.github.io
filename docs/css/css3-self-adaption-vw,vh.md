# css3自适应布局单位vw,vh

## 视口单位(viewport units) 
- 在pc端视口指的是浏览器的可视区域
- 而在移动端,它涉及3个视口: Layout View(布局视口)， Visual Viewport (视觉视口)
ldeal viewport (理想视口).
css3规范,视口单位主要包括以下4个:
1. vw: 1vw = 视口宽度的1%;
2. vh: 1vh等于视口高度的1%;
3. vmin: 选取vw和vh中最小的那个;
4. vmax：选取vw和vh中最大的那个;
vh和vw是相对于视口的高度和宽度,而不是父元素的(css百分比是相对于包含它的最近的父元素的高度和宽度). 1vh等于1/100的视口高度,1vw等于1/100的视口宽度。

## vh/vw与%区别
- % 百分比，相对长度单位，相对于父元素的百分比值
